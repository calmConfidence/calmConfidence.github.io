说明

持续更新中…

由于博客整体上可玩性很高，但是时间有限所以以主要功能为主，秉持初心，博客的核心是内容，对于其他功能用到再完善。😎🕶️

博客加密功能

详情见开源网址：hexo-blog-encrypt 这里简单总结一下：

  1. 安装包：npm install --save hexo-blog-encrypt–save 是指运行时的依赖，及上线后仍然需要该依赖

  2. 最简单使用😆: 直接在文章头中加入password选项即可

  3. 高级设置👍

  4. 配置文章信息头



2. 配置_config.yml文件（**注意：信息头的优先级高于配置文件**）

  1. 配置加密主题🤔: theme: (default、blink、shrink、flip、up、surge、wave、xray）

  2. 如果配置了tag加密，在某篇想解除，直接在信息头设置password: ""即可

  3. 注意：

    1. 配置了标签加密，需要hexo clean清理掉缓存才能使对应标签产生加密效果

    2. 如果在信息头添加了加密信息，则其余如wrong_pass_message的信息将不会应用_config.yml文件中的配置

  4. 遇到一些关于callback函数问题❓直接访问官网解决

博客本地搜索功能

这里使用的是hexo-generator-search插件，注意，当给文章加密后，搜索时是会避免搜索加密文章的。

默认情况，搜索图标是在最左边的，通过调整导航栏的配置文件可以实现修改到右边

路径：blog\themes\butterfly\layout\includes\header\nav.pug

添加访客位置信息功能

这里参考“铭心石刻”的方案，采用腾讯位置的api，相应教程如下：在侧边栏添加个性定位欢迎信息 | 铭心石刻 (kouseki.cn)

这里唯一的坑点：腾讯位置api注册完后并不能直接使用，需要手动去配置额度后才能正常使用。同时记得添加域名白名单，本博客白名单为：calmconfidence.github.io没有前面的https://

标记一下腾讯位置的网页：腾讯位置key
