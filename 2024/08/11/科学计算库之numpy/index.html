基本概念

  1. 什么是numpy？numpy用来干什么的？

    用于快速处理任意维度的数组，速度显著优于直接使用python来处理。

  2. 什么是ndarray?

    NumPy提供了一个N维数组类型ndarray，它描述了相同类型的“items”的集合。一种新的数据类型，专门用于处理多维数组运算的。

  3. ndarray的优势是什么？

    • 内存块风格

    • ndarray支持并行化运算（向量化运算）

    • ndarray底层是用C语言写的,效率更高,释放了GIL

  4. ndarray的属性？

    ndarray.shape 数组维度的元组
    ndarray.ndim 数组维数
    ndarray.size 数组中的元素数量
    ndarray.itemsize 一个数组元素的长度（字节）
    ndarray.dtype 数组元素的类型

正太分布 $\mu$代表均值 $\sigma$ 代表标准差，当$\mu$为0，$\sigma$为1时为标准正太分布，标准差反映了数据的离散程度，标准差越小分布越集中。

均匀分布是关于定义在区间[a,b], (a<b)上连续变量的简单概率分布。

数组的索引和切片

多维的本质上就是一种多个分组，比如说：当为一维时，我们通过每一列来寻找一个确定的数；当为两维时，原来只有一行，现在有多行，我们先确定行，再确定每个列中具体元素；当为三维时，原来的两维被装进了不同的袋子里，现在我们先选定是哪个袋子，再在这个袋子里去确定行，列来找到元素。当为四维时，原来的袋子被装进一个箱子，我们首先选定哪个箱子，再在这个箱子中去选定袋子，以此类推…

其他

  • 改变形状：array.reshape() array.resize()前一个不改变array本身，后一个改变array

  • 数组转置：array.T

  • 数组去重：np.unique(array)不管是几维的，最后统一降成一维的并把所有数组中的数据相同的去掉

数组与数组的运算

当满足下面两个条件时，数组与数组间维度不同会发生广播机制。

  1. 两个张量都必须不为空；

  2. 将两个张量的形状信息右对齐，从右向左观看同一维度下，张量的轴长必须相等，或者其中一个轴长为1，或者其中一个轴长为空。

矩阵

矩阵和数组之间的区别就是，矩阵必须是二维的，而向量不限制。尽管行向量看起来像一维，但仍是二维 例如：[1, 3] -> matrix = [1, 2, 3] matrix[0] = [1, 2, 3]

矩阵乘法

np.dot(a, b) np.matmul(a, b)前一个既可以进行矩阵和矩阵之间乘法也可以进行矩阵与标量之间乘法，后一个只能进行矩阵与矩阵之间乘法。
