机器学习概述

|字数总计:774|阅读时长:2分钟|阅读量:

简要复习

本篇内容主要是对机器学习初步知识的概述。

人工智能概述

人工智能发展必备三要素数据 算法 计算力（算力），其中算力分为cpu(central processing unit)计算gpu(graphics processing unit)计算tpu计算，cpu主要适合I/O密集型任务，gpu主要适合计算密集型任务。cpu和gpu的区别 简单来说gpu的计算单元远远超过cpu，且排布更“并行”，所以gpu很利于处理易于并行的程序。tpu是Google研究的一种ai芯片，Google tpu

人工智能和机器学习、深度学习的关系

机器学习是人工智能的一个实现途径，深度学习是机器学习的一个方法发展而来的。

发展历程

图灵测试：超过30%的测试者不能确定被测试者是人还是机器，则这台机器被认为是具有人工智能的。

达特茅斯会议：将该会议的研究内容定义为“人工智能”，1956年被定为人工智能元年。

人工智能主要分支

计算机视觉（CV）、自然语言处理（NLP）、机器人

自然语言处理包括 语音识别 文本挖掘/分类 机器翻译

机器学习的工作流程

从数据中自动分析获得模型，并利用模型对未知数据进行预测。

  1. 获取数据：样本（一行数据），特征（一列数据），划分训练集、测试集，目标值特征值是否成对出现

  2. 数据基本处理：处理缺失值、异常值

  3. 特征工程：使用专业背景知识和技巧处理数据，使得特征能在机器学习算法上发挥更好的作用的过程。

    1. 特征提取
    2. 特征预处理
    3. 特征降维

  4. 机器学习（模型训练）

  5. 模型评估

机器学习算法分类（依据数据集组成不同）

  1. 监督学习：输入数据是由 特征值和目标值组成

  2. 无监督学习：输入数据只有特征值，无目标值

  3. 半监督学习：输入数据同时包含标记样本数据集和未标记样本数据集

  4. 强化学习：实质是make decisions 问题，即自动进行决策，并且可以做连续决策。同时伴有奖励，目标是累计获取最大奖励。下棋就是一个典型的强化学习例子。

模型评估

分类模型评估、回归模型评估。过拟合、欠拟合

深度学习

深度学习是机器学习下的一个分支，主要利用的是神经网络，整体上分为 输入层、隐藏层、输出层。其中隐藏层包含多个层次，每一层负责特定的特征提取，而每一层又由多个神经元组成，这些神经元互不干扰完成该层所需完成的工作的某个部分。

一般来说，神经网络层数越多，提取的特征就更多，效果会更好，但绝不是层数越多越好。
