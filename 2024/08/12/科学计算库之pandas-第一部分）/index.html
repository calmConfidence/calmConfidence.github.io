科学计算库之pandas(第一部分）

pandas简介

pandas是一个综合了numpy和matplotlib两个库的科学计算库，专门用于数据挖掘。且其在数据表现效果上更加美观，且封装了文件读写的api，对一些缺失数据的处理也更好。

pandas的数据类型

Series

Series是一个类似与一维数组的数据类型，由索引和具体值两部分组成。

Series的相关属性，index values

DataFrame

DataFrame是一个类似于二维数组或表格的对象，既有行索引又有列索引。

常见属性和方法：shape、index、columns、values、T head() tail()

修改索引：data.index = new_index 这里修改索引只能整体修改，不可局部修改

重设索引：data.reset_index(drop=True) 当为True时，则删除原来索引，为False则保留原来索引。

以某一列值设为新的索引：data.set_index(keys, drop=True)

基本数据操作

  1. 索引操作

    1. 直接进行选择：遵循一个先列后行 data[columns][index]
    2. 通过.loc，然后根据索引名称，先行后列进行选择
    3. 通过.iloc，然后根据索引数字，先行后列进行选择

  2. 赋值操作：data[‘month’] = 12 data.month = 12

    1. 对DataFrame的赋值操作只能整体赋值，不能精确到索引来赋值

  3. 排序操作

    1. DateFrame：
      1. 值排序：data.sort_values(by=‘keys’, ascending=True) ascending=True升序 False降序
      2. 索引排序：data.sort_index()
    2. Series:
      1. 值排序：data.sort_values(ascending=True) 只有一列不需要keys
      2. 索引排序：data.sort_index()

DateFrame的运算

算术运算

  1. 直接使用 + - * \

  2. 使用对应方法（推荐）：data.add() data.sub() data.mul() data.div()

逻辑运算

  1. 逻辑符号> < >= <= == !=，通过使用这些逻辑符号，会返回对应数据位置的bool值

  2. 可以根据这个bool值取出对应数据

  3. 逻辑运算函数

    1. data.query(expr)

    2. data.isin(values)

  4. 统计运算

    1. 总体的统计结果：data.describe() 计算数量、均值、标准差、最大值、最小值以及一些四分位、二分位等

    2. 一些统计函数（参数为axis，默认为0按行统计）

      函数 作用
      sum Sum of values
      mean Mean of values
      median Arithmetic median of values
      min Minimum
      max Maximum
      count Number of non-NA observations
      abs Absolute Value
      prod Product of values
      std Bessel-corrected sample standard deviation
      var Unbiased variance（方差）
      idxmax compute the index labels with the maximum
      idxmin compute the index labels with the minimum
      mode Mode（众数）

    3. 累计统计函数

      函数 作用
      cumsum 计算前1/2/3/…/n个数的和
      cummax 计算前1/2/3/…/n个数的最大值
      cummin 计算前1/2/3/…/n个数的最小值
      cumprod 计算前1/2/3/…/n个数的积

  5. 自定义运算 data.apply(func, axis=0, 1) 只能在dataFrame对象上用

Pandas画图

pandas画图封装了matplotlib，具体使用直接查看api文档即可。

DataFrame.plot(kind=‘line’)这里的kind指定了画图的种类，这里就是折线图。

文件读取与存储

  • CSV文件

    • 读取：pd.read_csv(path, sep, usecols)

    • 存储：data.to_csv(path, sep, columns, header, index, mode)

  • HDF5文件

    • 读取：pd.read_hdf(path, key=) 读取时须制定key键一般说明文档会提供，如果该hdf5只有一个键可省略。
    • 存储：data.to_hdf(path, key=)存储时必须给定key键
    • 其他说明：pandas默认采用这种方式进行读写，因为这种方式支持压缩且跨平台，能够节省空间。

  • JSON文件

    • 读取：pd.read_json(path, orient, typ, lines)

    • 存储：data.to_json(path, orient, lines)
